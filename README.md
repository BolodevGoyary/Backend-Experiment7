# Backend-Experiment7: Search Functionality

## Steps

### 1. Use Existing Database
Use the same `backend_experiments` database and `posts` table from previous experiments.

### 2. Place Files
Put all files in:
MAMP: /Applications/MAMP/htdocs/experiment7  

Files:
- db_connect.php
- search.php

### 3. Run
MAMP:  
http://localhost:8888/experiment7/search.php  

### 4. Search
Enter a keyword in the search bar and submit to filter posts by title.
